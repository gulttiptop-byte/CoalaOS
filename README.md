# 🐨 Coala OS v9.1 "Evolution"

**Coala OS** — это легковесная имитация операционной системы, написанная на Python с использованием графической библиотеки PyQt5. Проект включает в себя рабочий стол, систему окон (MDI), браузер, текстовый редактор и встроенного AI-ассистента.

## ✨ Основные возможности
* 🖥️ **Рабочий стол с сеткой:** Иконки можно перетаскивать, они автоматически выравниваются.
* 🖼️ **Персонализация:** Возможность устанавливать свои обои (.png, .jpg).
* 🤖 **AI Chat:** Интегрированный помощник через API OpenRouter.
* 🛠️ **Coala IDE:** Собственная среда разработки. Создавайте программы `.coef` (исполняемые) или заметки `.coif`.
* 🌐 **Браузер:** Полноценный просмотр веб-страниц (требует WebEngine).
* 📂 **Проводник и Терминал:** Управление файлами внутри папки `apps`.

### 1. Требования
Убедитесь, что у вас установлен **Python 3.8** или выше.
